# SQLD 학습앱

SQLD 자격증 시험 대비용 플래시카드 형태의 단일 HTML 학습 앱입니다.

## 사용법

`sqld_study.html` 파일을 브라우저에서 열면 바로 실행됩니다. 별도 설치 불필요.

## 구성

| 과목 | 단원 |
|------|------|
| 1과목 | 데이터 모델링의 기본 / 엔터티와 속성 / 관계와 식별자 / 정규화와 성능 |
| 2과목 | SQL 기본 / JOIN과 서브쿼리 / 그룹·윈도우 함수 / SQL 최적화 기본 / TOP N·계층형 질의 / 정렬·정규표현식 |

## 기능

- 카드 앞면(개념) → 탭하면 뒷면(예시 코드 + 핵심 요약) 확인
- 이전 / 다음 버튼으로 카드 탐색
- 중요 카드 저장 후 **요약노트**에서 메모 편집 가능
- 진행도 바로 학습 현황 파악

## 기술 스택

순수 HTML / CSS / JavaScript — 외부 라이브러리 없음
